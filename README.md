    - `[”Hi”, “Mama”]`
    - `[3.14, 2.88]`

- Functions definitions
    ```jsx
    function function_name(parameter1, parameter2){
    	//function body
    	
    }
    `````
    
- Branches
    - if/else
    
    ```jsx
    if (test_condition){
    	//code block
    } else{

    } 
    ```
    
    - if
    
    ```jsx
    if (test_condition){
    	//code block
    }
    ```
    
- Loops
    - For Loop
    
    ```jsx
    const my_list = ["Yammani", "Murari", "Sunny"]
    for (var i = 0; i < my_list.length; i++) {

    }
    
    for (var item in my_list){

    }
    ```
    
    - While Loop
    
    ```jsx
    while (var i < test_condition){
    	//code block
    }
    ```
